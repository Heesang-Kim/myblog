블로그를 만들면서 **이런 기술들**을 썼어요 🛠️

## 사용한 재료 🧁

- **Next.js 16** — App Router로 페이지를 슥슥
- **React 19** — 요즘 핫한 최신 버전
- **Tailwind CSS 4** — 귀여운 파스텔 톤을 입혔어요
- **Markdown + gray-matter** — 글은 간단하게 `.md` 파일로 관리

## 특히 좋았던 점

```tsx
export default function Page() {
  return <p>정말 간단해요 🌼</p>;
}
```

서버 컴포넌트로 글 목록을 바로 읽어올 수 있어서 편했어요.
파일 하나 추가하면 바로 블로그 포스트가 되니까, 글 쓰는 리듬이 안 끊기더라구요.
